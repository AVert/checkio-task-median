Медиана это числовое значение, которое делит ранжированный массив чисел на большую и меньшую половины. В массиве, где нечетное число элементов, медиана это число в середине ранжированного массива. Для массива с четным числом элементов, где нет одного элемента точно в середине, медиана это среднее значение двух чисел находящихся в середине ранжированного массива. В этой задаче, дан не пустой массив натуральных чисел. Вам необходимо найти медиану в данном массиве.

Вх. данные: Массив как список (list) чисел (int).

Вых. данные: Медиана как число (int, float).

Примеры: 

checkio([1, 2, 3, 4, 5]) == 3
checkio([3, 1, 2, 5, 3]) == 3
checkio([1, 300, 2, 200, 1]) == 2
checkio([3, 6, 20, 99, 10, 15]) == 12.5

Как это используется: Медиана находит свое применение в статистике и теории вероятности, Особенно это важно для ассиметричного распределения. Для примера: мы хотим узнать среднее доход населения -- 100 человек получают $100 в месяц и 10 человек получают $1,000,000. Если мы возьмем среднее арифметическое, то получим $91,000. Достаточно странное число и не показывает истинное положение дел. В этом случае медиана будет для нас более полезной величиной и покажет более правдоподобную картину. Статья на ВикипедиЯ.

Предусловия:
1 < len(array) ≤ 1000
∀ x ∈ X : 0 ≤ x < 106
